GITHUB REPO SYNC - WINDOWS PC SETUP
====================================

Follow these steps to edit this repo on your Windows PC:

1. INSTALL PREREQUISITES
   - Download and install Git for Windows: https://git-scm.com/download/win
   - Download and install VS Code: https://code.visualstudio.com/
   - Both should be on your PATH (default installation is fine)

2. OPEN VS CODE
   - Launch VS Code
   - Install the "GitHub Pull Requests and Issues" extension
     * Ctrl+Shift+X
     * Search for "GitHub Pull Requests and Issues"
     * Click Install

3. SIGN IN TO GITHUB
   - Ctrl+Shift+P
   - Type "GitHub: Sign in"
   - Follow the prompts to authenticate

4. CLONE THE REPO
   - Ctrl+Shift+P
   - Type "Git: Clone"
   - Paste: https://github.com/basukabomberumpe-design/donttellyourteachers.github.io.git
   - Choose a folder (e.g., Documents\GitHub)
   - Wait for the clone to finish
   - Click "Open" when prompted

5. SET UP GIT CONFIG
   - Open the terminal in VS Code: Ctrl+`
   - Make sure the terminal says "Git Bash" (bottom right)
     * If it says "PowerShell", click the dropdown and select "Git Bash"
   - Run these commands:
     git config user.name "basukabomberumpe-design"
     git config user.email "basukabomberumpe@gmail.com"

6. YOU'RE READY TO CODE!
   - Edit any files in the repo
   - Save your changes (Ctrl+S)
   - Open Git Bash terminal: Ctrl+`
   - Run: ./sync.sh
   - This will commit and push your changes to GitHub

7. SYNC WITH OTHER PC
   - On your Chromebook (or other PC): run "git pull" to get your Windows changes
   - On your Windows PC: run "git pull" to get changes from other PCs

TROUBLESHOOTING
   - If "./sync.sh" doesn't work, make sure you're in Git Bash (not PowerShell)
   - If auth fails, you may need a GitHub Personal Access Token (PAT)
     * Generate one at: https://github.com/settings/tokens
     * Use it as your password when prompted
   - If you get merge conflicts, ask for help!

QUICK WORKFLOW
   1. Edit files
   2. Save (Ctrl+S)
   3. Run ./sync.sh in Git Bash
   4. Done! Changes are live on GitHub
